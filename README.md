# Brew formula

## Tapping

```shell
brew tap cgaube/dev-commands
```
## Installing formula

### Main `dev` utils

```shell
brew install --HEAD dev-commands
```

## Upgrading formula 

Because we are using head mode we need do a `reinstall`

```shell
brew reinstall dev-commands
```

## Updating homebrew-formulas

- edit formula files
- commit code

```shell
brew update
```
